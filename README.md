## Client Setup
When rebranding this static site for a new plumbing client, update:
- **Business name & tagline** across headings, logo alt text, and footer.
- **Service area** references (city/town names).
- **Phone number** in all `tel:` links and visible numbers.
- **WhatsApp link** (`wa.me` number and pre-filled message).
- **Contact email & address (optional)** in Privacy/Terms and any forms.
- **Call-out/diagnosis fee policy** and amount if applicable.
- **Warranty period** for workmanship (Terms).

- **Analytics ID** if adding GA or similar; remove console-only tracking when replaced.
